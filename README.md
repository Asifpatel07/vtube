This is a clone of Youtube app , I have bulit many features which Youtube has it, and I have named it as Vtube. You can name it anything you with to. Keep checking out for more new features in it.

I have use Reactjs, TailwindCss, Redux, React-router-dom to build this.

Build features like:
1. Youtube real time api for listing down the data on home page
2. Search suggestions from api with dibouncing feature
3. Cacheing the searched suggestions in redux store
4. Live chat data for live streaming
5. watch page
6. Comments section for specific video on watch page
7. Collapsible Sidebar
